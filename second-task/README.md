### Установка и запуск клиента
- Установка зависимостей ```npm install```
- Запуск проекта в ```dev ``` режиме ```npm run start```
- Сборка проекта в ```prod``` режиме ```npm run build```
###Установка и запуск сервера
- ```cd backend```
- Установка зависимостей ```npm install```
- Запуск сервера ```node server.js```
